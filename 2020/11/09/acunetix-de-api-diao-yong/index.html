2.筛选目标信息

参数 类型 说明
threat int 威胁等级;高->低:[3,2,1,0]
criticality int 危险程度;高->低:[30,20,10,0] 
Method:GET 
URL: /api/v1/targets?q=threat:{list};criticality:{list};

/api/v1/targets?q=threat:3;criticality:10,20

3.添加目标

Method:POST 
URL: /api/v1/targets

这要发一个POST请求
data:

{"address":"http://www.sqlsec.com","description":"xxxx","criticality":"10"}
参数 类型 说明
address string 目标网址:需http或https开头
criticality int 危险程度;高->低:[30,20,10,0];默认为10
description string 备注
代码 
import requests ,json
from requests.packages.urllib3.exceptions import InsecureRequestWarning

requests.packages.urllib3.disable_warnings(InsecureRequestWarning)

api_url = 'https://127.0.0.1:3443/api/v1/targets'
headers = {
    'X-Auth': '1986ad8c0a5b3df4d7028d5f3c06e936c2561802e7fca4687b81de68875a05bfb',
    'Content-type': 'application/json'
}
d = {"address":"http://www.sqlsec.com","description":"python测试","criticality":"10"}
r = requests.post(url=api_url, data=json.dumps(d), headers=headers, verify=False)
print(r.json())

返回的数据
